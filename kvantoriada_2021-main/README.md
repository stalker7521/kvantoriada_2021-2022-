## More information
- To work MPU-9250 without errors, you need to initialize both the accelerometer and the magnetometer.  
mpu = MPU9250()  
mpu.initialize()  
mpu.gyro_offs = {'y': -5, 'x': 158, 'z': -100}  
mpu.accel_offs = {'y': 102, 'x': -34, 'z': -364}

compas.initialize()
